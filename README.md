# 3D Design Drone

Desain ini bertujuan untuk menghasilkan frame drone yang kuat, ringan, dan mudah untuk dirakit.

## Fitur
- Desain base atas
- Desain base bawah
- Sayap
- Kaki

## Tools
- Software Autodesk Inventor

## Cara Menggunakan
1. Buka file desain di software inventor
2. Export ke file STL jika ingin dicetak
3. Cetak menggunakan 3D printer
4. Rakit komponen drone sesuai desain

## Tujuan
- Melatih kemampuan desain 3D
- Membuat drone yang efisien
- Mengembangkan proyek di bidang robotika
